# TicTacToe-AI

A tic-tac-toe game AI written in c/c++.

## Demos

![](img/preview.png)

## Compile and Run

Windows:

    Compile and run with Visual Studio or g++.
    
Linux:

```bash
$ cd src
$ make
$ make run
$ make clean
```
